OsteoVision — 膝関節X線ポジショニングAI

AI-powered Knee X-ray Angle Estimation & Positioning QA System

【概要】 膝関節X線画像からYOLOv8-poseでキーポイントを検出し、関節角度・回旋量を自動推定。ポジショニング品質を定量的にフィードバックするシステム。

  • 合成DRR自動生成: OsteoSynthで720枚の訓練データを生成（患者データ不要）
  • 精度: mAP50 = 99.8%（633枚で訓練）
  • XAI: Grad-CAMによる注視領域可視化
  • CI/CD: GitHub Actions + Docker

ElbowVision — 肘X線ポジショニングAIガイド

AI Positioning Guidance System for Elbow X-ray Imaging

【概要】 肘X線画像から外反角（Carrying angle）・屈曲角をAIで推定し、ポジショニング補正アドバイスをリアルタイム提供するシステム。

  • 実ファントムX線使用: ドメインギャップ問題を根本解決
  • 倫理委員会不要: ファントム（人体模型）のみで研究可能
  • デュアルモデル: YOLOv8-pose（主）+ ConvNeXt（セカンドオピニオン）
  • エッジバリデーション: Canny + HoughLinesPで骨軸を独立検出

ConvNeXt-Tiny Grad-CAM TTA

膝関節X線撮影における再撮影判断支援のための深層学習プログラムの開発

Deep Learning Program for Retake Decision Support in Knee X-ray Imaging

ConvNeXt-Tinyを用いて膝関節側面像の角度ズレを自動推定。R² > 0.96、MAE 約1〜2°を達成。Grad-CAMにより熟練技師と同様の注視領域（大腿骨顆部）を学習していることを確認。現在のOsteoVision（YOLOv8-pose版）の前身研究。

  • Model: ConvNeXt-Tiny (ImageNet-1k Pre-trained)
  • Data: Phantom simulation dataset (671 images)
  • R² > 0.96 / MAE: 1〜2°
